# A Multi-Technique Approach for Improving Summary Polar Diagrams - Interactive Demo Video

## Description
In this video, viewers can expect to see key features, user interactions, and visualizations present in the dashboard that is presented in the main paper. This demonstration aims to provide insights into how the multi-technique approach is utilized for improving summary polar diagrams in two example applications covered in the paper (Climate Model Comparison and Machine Learning Hyper-parameter Tuning).
