# A Multi-Technique Approach for Improving Summary Polar Diagrams - Interactive Demo Video

## Description
This repository contains a video file that demonstrates the interactive capabilities of the dashboard created for the manuscript named "A Multi-Technique Approach for Improving Summary Polar Diagrams". In this video, viewers can expect to see key features, user interactions, and visualizations present in the dashboard. This demonstration aims to provide insights into how the multi-technique approach is utilized for improving summary polar diagrams.
